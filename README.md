OpCheck
========
####x86Check
Scan binaries for various x86 opcodes.

Supported:
* Pentium
* Pentium MMX
* AMDK6
* Pentium Pro
* SSE
* SSE 2
* SSE 3
* SSE 4.1
* SSE 4a
* SSE 4.2
* Intex AVX FMA
* x86_64
* AMD-V
* Intel VT-x
* AMD 3DNow!
* AMD 3DNow!+
* ABM
* BMI1
* BMI2
* TBM

####armCheck
Scan binaries for various arm opcodes.

Supported:
* Intel Wireless MMX
* Floating point coprocessor (FPA)
* Floating point coprocessor (VFP)
* NEON
* Cirrus coprocessor
